# Base62 Utility for Java

An optimized, standalone Java utility class for encoding base-10 `long` integers into Base62 strings and decoding Base62 strings back into `long` integers based on the Euclidean Division Algorithm for Radix Conversion.

This implementation is specifically designed for maximum throughput and minimal memory footprint, making it ideal for high-performance applications such as URL shorteners, distributed ID generators, and database key conversions.

## Usage

This utility is a single, dependency-free Java class. 

**The best practice for using this library is to simply copy and paste the `Base62.java` source code directly into your project.**
